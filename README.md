# 迷宫探险游戏

PBLF课程项目 - 使用C语言和Raylib框架的迷宫探险游戏

##  游戏简介
一个迷宫探险游戏，玩家需要探索迷宫、躲避NPC、收集道具，最后找到出口。

##  核心功能
-  迷宫系统（随机生成）
-  玩家控制系统（鼠标移动）
-  NPC巡逻与挑战
-  实时动态事件（道具刷新）
- 可视化UI（生命状态提示）

##  技术栈
- **语言**: C语言
- **图形框架**: Raylib 
- **开发环境**: VS Code 
- **版本控制**: Git + GitHub
